## 🎯 Features

- 🔍 Predict diabetes risk with patient details
- 📊 Interactive data visualizations
- 🧬 8 medical features used for prediction
- 📈 Probability score with risk breakdown
- 💡 Health recommendations based on results

---

## 🛠️ Tech Stack

| Tool | Purpose |
|---|---|
| Python | Core Language |
| Scikit-learn | ML Model |
| Pandas & NumPy | Data Processing |
| Streamlit | Web App |
| Matplotlib & Seaborn | Visualizations |
| SHAP | Model Explainability |

---

## 📊 Dataset

- **Pima Indians Diabetes Database** (Kaggle)
- 768 patients, 8 features
- Binary classification — Diabetic / Not Diabetic

---

## 🤖 Model Performance

| Model | Accuracy |
|---|---|
| Logistic Regression | ~75% |
| Random Forest | ~77.92% ✅ Best |
| Gradient Boosting | ~76% |
| SVM | ~74% |

## ▶️ Run Locally

```bash
git clone https://github.com/Veeruchalla88/diabetes-prediction-ml.git
cd diabetes-prediction-ml
pip install -r requirements.txt
streamlit run app.py
```
